This project limited to get data from database using Chakra UI as UI Library

First step, after you download you can 

### `npm install`

and then run with

### `npm start`

go to server folder

### `cd server`
and
### `npm install`
to run
### `node index.js`

and the app will run in your browser http://localhost:3000
